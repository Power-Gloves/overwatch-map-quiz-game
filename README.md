# 守望先锋地图识别游戏

一个基于Vue 3 + TypeScript的守望先锋地图识别游戏，支持多种游戏模式和智能难度调节。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 游戏模式

- **练习模式** (10题): 显示完整地图名称，适合新手
- **挑战模式** (20题): 智能文本模糊，需达到70%正确率通关  
- **无尽模式**: 无限题目，追求连续答对记录

## 技术栈

- Vue 3 + TypeScript
- uni-app (跨平台)
- VueBits组件库
- SCSS + 守望先锋UI风格
