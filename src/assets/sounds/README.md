# 游戏音效文件说明

## 音效文件列表

请将以下音效文件放置在此目录中：

### UI音效
- `button-click.mp3` - 按钮点击音效
- `button-hover.mp3` - 按钮悬停音效  
- `card-flip.mp3` - 卡片翻转音效
- `card-swipe.mp3` - 卡片滑动音效

### 游戏音效
- `correct-answer.mp3` - 答对音效
- `wrong-answer.mp3` - 答错音效
- `game-start.mp3` - 游戏开始音效
- `game-win.mp3` - 游戏胜利音效
- `game-lose.mp3` - 游戏失败音效

### 背景音乐
- `background-music.mp3` - 游戏背景音乐
- `menu-music.mp3` - 菜单背景音乐

## 音效要求

- **格式**: MP3 (推荐) 或 WAV
- **时长**: 
  - UI音效: 0.1-0.5秒
  - 游戏音效: 0.5-2秒
  - 背景音乐: 30秒-2分钟 (循环播放)
- **音质**: 44.1kHz, 16-bit 或更高
- **文件大小**: 尽量控制在合理范围内以优化加载速度

## 音效风格建议

- **守望先锋风格**: 科技感、未来感的音效
- **UI音效**: 清脆、简洁的提示音
- **正确答案**: 积极、鼓励的音效
- **错误答案**: 温和的提示音，不要过于刺耳
- **背景音乐**: 轻松、不干扰游戏的氛围音乐

## 使用方法

音效系统已经集成到游戏中，会在以下时机自动播放：

1. **按钮交互**: 点击和悬停时
2. **答题反馈**: 根据答案正确性播放不同音效
3. **游戏状态**: 开始、结束时的音效
4. **背景音乐**: 根据游戏状态切换不同音乐

## 音量控制

- 用户可以通过右下角的静音按钮控制音效
- 音效和音乐音量可以分别调节
- 支持完全静音模式
